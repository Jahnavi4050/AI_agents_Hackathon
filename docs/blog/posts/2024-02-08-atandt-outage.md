# AT&T Nationwide Outage

## Summary
On February 8, 2024, a core network failure caused AT&T's cellular and internet services to go down nationwide, affecting millions of customers.

## Timeline
- **February 8, 2024, 6:30 AM EST**: Issue detected in core network.
- **Morning**: Widespread outages reported.
- **Afternoon**: Partial restoration.
- **Evening**: Most services back online.

## Root Cause
A misconfiguration in the core routing system led to a failure that cascaded across the network.

## Impact
- Affected ~70,000 customers.
- Disrupted calls, texts, and data services.
- Emergency services impacted in some areas.

## Resolution
- Network reconfiguration and rerouting.
- Investigation into the misconfiguration.

## Lessons Learned
- Redundancy in core network components.
- Faster detection and response.
- Better configuration management.
