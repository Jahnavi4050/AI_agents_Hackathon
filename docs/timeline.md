# Internet Outage Timeline

This interactive timeline visualizes major internet outages throughout history, from the first computer worm in 1988 to recent global service disruptions. Each entry represents a significant incident that affected internet reliability or availability.

## 游닀 How to Use This Timeline

- **Chronological Order**: Events are organized from newest to oldest
- **Color Coding**: Different colors represent different types of incidents
- **Click to Explore**: Click "Read Report" on any entry to see the full technical analysis
- **Educational Value**: Learn from past failures to build more resilient systems

## 游늵 Timeline Legend

- 游댮 **Global Outages**: Affected millions of users worldwide
- 游리 **Regional Outages**: Impacted specific geographic areas
- 游 **Service-Specific**: Affected particular platforms or services
- 游댯 **Cyber Attacks**: Security-related incidents
- 游릮 **Infrastructure**: Network or hardware failures
