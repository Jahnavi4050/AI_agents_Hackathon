## üì§ Submit Missing Reports

If you have access to official postmortem reports for outages not listed here, please help expand this collection:

### How to Submit
1. **Find the Official Report**: Locate the official RCA/postmortem from the affected organization
2. **Download the Document**: Save as PDF or HTML format
3. **Submit via GitHub**: Create a pull request or open an issue with the report
4. **Include Metadata**: Date, organization, incident type, and source URL

### Report Requirements
- Must be from the official source (company blog, status page, etc.)
- Should contain root cause analysis and lessons learned
- Preferably in PDF format for consistency
- Include proper attribution and licensing

## üîç Report Missing Incidents

Know of a major internet outage that's not documented in this archive?

### Submit a New Incident
1. **Check Existing Reports**: Search to ensure it hasn't been documented
2. **Gather Information**: Date, affected services, impact, root cause if known
3. **Find Official Sources**: Links to news articles, status pages, or postmortems
4. **Submit Details**: Use the contribution form or GitHub issue

### What We Need
- Incident date and duration
- Services affected and geographic scope
- Estimated impact (users affected, economic cost)
- Root cause if available
- Links to official statements or postmortems
- Reliable news sources for verification
