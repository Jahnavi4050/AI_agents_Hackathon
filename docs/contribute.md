# Contribute to the Internet Outage Archive

We welcome contributions from the community to keep this archive comprehensive and up-to-date. If you've encountered or know of an outage not covered here, please help us document it.

## How to Contribute

### Option 1: Submit via GitHub
1. Fork this repository on GitHub
2. Create a new branch for your contribution
3. Add your postmortem report following the format below
4. Submit a pull request

### Option 2: Report Missing Outages
If you find an outage that's missing from our archive, please:
- Open an issue on our GitHub repository
- Include details about the outage (date, affected service, impact)
- Provide links to official postmortem reports if available

### Option 3: Suggest Improvements
- Report bugs or suggest UI/UX improvements
- Propose new features or analysis tools
- Help with translations or accessibility

## Postmortem Report Format

Please follow this structure for new reports:

```markdown
---
tags:
  - service-name
  - outage-type
  - other-relevant-tags
---

# [Service Name] Outage

## Summary
Brief overview of what happened.

## Timeline
- **Date/Time**: Event description
- **Date/Time**: Next event

## Root Cause
Technical explanation of what caused the outage.

## Impact
Who was affected and how.

## Resolution
How the issue was fixed.

## Lessons Learned
Key takeaways and improvements made.
```

## Guidelines
- Use factual information only
- Cite sources when possible
- Maintain neutrality and accuracy
- Follow the existing naming convention: YYYY-MM-DD-service-description.md

## Contact
For questions or suggestions, reach out via GitHub issues or discussions.

Thank you for helping preserve the history of internet outages!
